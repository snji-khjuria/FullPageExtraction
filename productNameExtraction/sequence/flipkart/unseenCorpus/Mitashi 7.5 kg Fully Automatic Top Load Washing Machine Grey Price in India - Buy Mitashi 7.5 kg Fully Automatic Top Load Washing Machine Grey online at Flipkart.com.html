    Product Description
    Get lint-free clothes that are protected from any damage while washing with Mitashiâ€™s fully automatic top-load washing machine. With the Intelligent Hall Sensor Advanced Program and the auto balance sensor, this washing machine adjusts the wash cycle according to your laundry load. It also maintains a healthy and bacteria-free environment for every wash.
    Intelligent Hall Sensor Advanced Program

    This program automatically adjusts the wash program, water level, wash time and rinse time according to the weight, fabric and dirtiness of the clothes.

    Magic Lint Filter

    This filter collects lint from clothes during a wash cycle in a lint filter bag, which can be emptied manually.

    Eco Healthy Wash

    To create a healthy and bacteria-free environment, this built-in program measures the sterilisation process during the wash cycle.

    Soak Function

    Often used as a pre-wash cycle, this function shuts off the machine to soak the clothes in water and detergent before the wash cycle.

    Overload Protection for Motors

    A small circuit automatically detects machine overload, protecting the motor from damage.

    Digital Display

    It comes with a digital board to display your current wash program and give alerts on the wash cycle time.

    Power Off Protection

    The washing machine automatically turns off post wash cycle, thus reducing the power consumption.

    Auto Balance Sensor

    The washing machine adjusts the wash program based on the load.

    Raindrop Safe Wash

    The Drop-in Drum Soft Protection technology for the inner drum ensures consistent flow of water, thus protecting the clothes from damage.

    Child Lock

    Particularly useful for homes with kids; this feature does not affect the current wash program even if other buttons are pressed accidentally.

    Buzzer

    This feature indicates the end of a wash cycle.
